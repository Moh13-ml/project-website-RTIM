## DÉPLOIEMENT LOCAL (DÉVELOPPEMENT ET TEST)

### Méthode 1: Ouverture directe dans le navigateur
1. Télécharger tous les fichiers du projet
2. Créer un dossier "RTIM -website" sur votre ordinateur
3. Placer tous les fichiers dans ce dossier
4. Double-cliquer sur "index.html" pour ouvrir dans le navigateur
5. Le site s'ouvrira à l'adresse: file:///chemin/vers/votre/dossier/index.html

**Limitations de cette méthode:**
- Certaines fonctionnalités JavaScript peuvent être limitées
- Pas de simulation d'environnement serveur
- Formulaire d'inscription fonctionne mais données stockées localement uniquement

### Méthode 2: Serveur local avec Python (RECOMMANDÉE)
**Prérequis:** Python installé sur votre ordinateur

1. Ouvrir le terminal/invite de commande
2. Naviguer vers le dossier du projet:
   ```
   cd chemin/vers/RTIM-website
   ```
3. Lancer le serveur local:
   - Python 3: `python -m http.server 8000`
   - Python 2: `python -m SimpleHTTPServer 8000`
4. Ouvrir le navigateur et aller à: http://localhost:8000
5. Pour arrêter le serveur: Ctrl+C dans le terminal

### Méthode 3: Serveur local avec Node.js
**Prérequis:** Node.js installé

1. Installer un serveur HTTP simple:
   ```
   npm install -g http-server
   ```
2. Naviguer vers le dossier du projet
3. Lancer le serveur:
   ```
   http-server -p 8000
   ```
4. Ouvrir: http://localhost:8000

### Méthode 4: Extension navigateur (Chrome/Firefox)
1. Installer l'extension "Live Server" ou "Web Server for Chrome"
2. Sélectionner le dossier du projet
3. Lancer le serveur local via l'extension

## OPTIMISATIONS AVANT DÉPLOIEMENT

### Performance
1. Minifier le CSS et JavaScript (optionnel pour ce projet)

### Sécurité
1. Configurer HTTPS (SSL/TLS)
2. Ajouter des headers de sécurité
3. Configurer les redirections www/non-www

## CONFIGURATION SERVEUR WEB

### Apache (.htaccess)
```
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

<IfModule mod_deflate.c>
    AddOutputFilterByType DEFLATE text/plain
    AddOutputFilterByType DEFLATE text/html
    AddOutputFilterByType DEFLATE text/xml
    AddOutputFilterByType DEFLATE text/css
    AddOutputFilterByType DEFLATE application/xml
    AddOutputFilterByType DEFLATE application/xhtml+xml
    AddOutputFilterByType DEFLATE application/rss+xml
    AddOutputFilterByType DEFLATE application/javascript
    AddOutputFilterByType DEFLATE application/x-javascript
</IfModule>
```

### Nginx
```
server {
    listen 80;
    server_name artem-mali.org www.artem-mali.org;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl;
    server_name artem-mali.org www.artem-mali.org;
    
    root /var/www/artem;
    index index.html;
    
    location / {
        try_files $uri $uri/ =404;
    }
    
    gzip on;
    gzip_types text/css application/javascript text/javascript;
}
```

## PROCÉDURES D'ADMINISTRATION

### Accès aux données candidats:
1. Ouvrir le site web
2. Appuyer sur F12 pour ouvrir la console
3. Utiliser les commandes artemAdmin.*

### Export des candidats:
1. Dans la console: `artemAdmin.exportData()`
2. Fichier JSON téléchargé automatiquement
3. Ouvrir avec Excel ou Google Sheets

### Recherche de candidats:
```javascript
// Rechercher par formation
artemAdmin.searchCandidates({formation: 'telecom'})

// Rechercher par nom
artemAdmin.searchCandidates({nom: 'Diarra'})

// Rechercher par période
artemAdmin.searchCandidates({
  dateDebut: '2025-01-01',
  dateFin: '2025-01-31'
})
```
